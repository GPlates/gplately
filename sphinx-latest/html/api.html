Main ClassesÔÉÅ

This page lists the main classes of the GPlately Python package.

Note

  • See the installation section for information on how to install GPlately.

  • See the examples section to help get you started using GPlately.

gplately.DataServer

Download plate reconstruction models to the cache folder on your computer from EarthByte's WebDAV server.

gplately.PlateReconstruction

Reconstruct topology features to specific geological times given a rotation_model, a set of topology_features and a set of static_polygons.

gplately.Points

Reconstruct and work with with geological point data.

gplately.Raster

The Raster class handles raster data.

gplately.PlotTopologies

Read, reconstruct and plot topology features at specific reconstruction times.

gplately.SeafloorGrid

A class to generate grids that track data atop global ocean basin points (which emerge from mid ocean ridges) through geological time.
