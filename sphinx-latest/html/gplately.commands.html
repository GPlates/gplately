BirthAgeFilter(500) – keep features whose time of apprearance are bigger than 500 BirthAgeFilter(500, keep_older=False) – keep features whose time of apprearance are smaller than 500

Parameters:

  • keep_older – if True, return True when the feature’s birth age is older than the age criterion. If False, otherwise.
